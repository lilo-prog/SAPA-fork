# SAPA
- A web application to help people with diseases. Some features include: community, receiving medical presciptions, health questionnaires, reminders, advices and the ability to communicate with doctors. Ambulatory Patient Accompaniment System (APAS).
- Una aplicación web para ayudar a personas con enfermedades. Algunas de sus funciones incluyen: comunidad, recepción de recetas médicas, cuestionarios de salud, recordatorios, consejos y la posibilidad de comunicarse con los médicos. Sistema de Acompañamiento para pacientes Ambulatorios (SAPA).
